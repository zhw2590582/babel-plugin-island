# babel-plugin-island

island plugin for Babel (目前测试用)

## Install

```
$ npm i --save-dev babel-plugin-island
```

## Usage

```js
// Add babel-plugin-island in .babelrc

"plugins": [
  [
    "island",
    {
      "author": "Harvey Zack",
      "email": "717995589@qq.com",
      "homepage": "http://www.zhw-island.com"
    }
  ]
];
```
